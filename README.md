# Adicion

## Enunciado

Mostrar por pantalla la suma de 2 numeros enteros ingresados por el usuario.

## Análisis

_a y b ∈ ℤ_

Entradas:

a ; b. 

Proceso:

Sumar a+b.

Salida:

Resultado de sumar a+b.

## Algoritmo

_a y b ∈ ℤ_


|    Inicio         |
|:-----------------:|
|    **Leer a**     |
|    **Leer b**     |
|    **Mostrar a+b**|
|    **Fin**        |
